    Any page can be presented as a modal dialog by adding the data-dialog="true" attribute to the page. When the "dialog" attribute is applied,
  1. the framework adds styles to add rounded corners, margins around the page and a dark background to make the "dialog" appear to be suspended above the page.
  2. By default the framework will also add a close button if the dialog has a header.
